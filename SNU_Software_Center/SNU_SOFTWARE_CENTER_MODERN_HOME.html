SNU software development center (modern)

This page is the hub for the development of SNU software, and the download of non-game, non-malware software. For the malware section, refer to the SNU malware center, for the gams section, refer to the SNU Games center.

This page is intended for avid and professional computer users and software developers, however anyone can join if they can contribute, or if they are looking for a particular piece of software. Discussion is also allowed.

Games are not allowed here, for games, try out the SNU game player here
